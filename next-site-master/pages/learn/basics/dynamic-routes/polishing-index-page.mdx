Next, let’s update our index page (`pages/index.js`).
We need to add links to each post page using the [`Link`](/docs/api-reference/next/link) component.

Let's import
[`Link`](/docs/api-reference/next/link) and `Date` in `pages/index.js`:

```js
import Link from 'next/link'
import Date from '../components/date'
```

Then, near the bottom of the `Home` component, replace the `li` tag with the following:

```jsx
<li className={utilStyles.listItem} key={id}>
  <Link href={`/posts/${id}`}>
    <a>{title}</a>
  </Link>
  <br />
  <small className={utilStyles.lightText}>
    <Date dateString={date} />
  </small>
</li>
```
